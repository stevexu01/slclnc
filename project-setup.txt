1. Project in Github:
	https://github.com/stevexu01/slclnc.git

2. Setup:
	(1) Download and install MySQL database
	https://dev.mysql.com/downloads/windows/installer/5.7.html
	install instruction - https://blog.zedfox.us/install-mysql-5-7-windows-10/
	** set mysql root password to "ginger"

	(2) Start MySQL server
	In above instruction.

	(3) Clone source code:
		mkdir <your_workspace>
		git clone https://github.com/stevexu01/slclnc.git
		cd slclnc

	(4) Create database and tables:
	slclnc > mysql -uroot
	mysql > create database clinic
	mysql > use clinic
	mysql > source <your_workspace>/slclnc/doc/dbcreate.sql


	(5) Start application
	 ** from command line:
	 > mvn clean jetty:run

	 ** from IDE

	(6) Access application:
	http://localhost:8080
